# alenliu.space · Personal Landing Page

A minimal, Greyforge-style personal landing page for Alen Liu, focused on
global expansion, cross-border e-commerce, and TikTok Shop operations.

## Structure

- `index.html` – main page
- `assets/css/style.css` – layout & visual styles
- `assets/js/main.js` – small enhancements (e.g. dynamic year)

## How to Run Locally

Just open `index.html` in your browser.

Or use a static server, for example:

```bash
python3 -m http.server 4173
